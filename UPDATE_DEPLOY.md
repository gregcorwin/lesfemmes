# Update & Deploy

This project auto-deploys to Vercel on every push to `main`.

## Make a change
- Edit `lesfemmesv11.html` (and any assets).

## Commit and push
From the project folder:

```
git status -sb
git add .
git commit -m "Update content"
git push
```

## Confirm
- Vercel will deploy automatically.
- Check `https://lesfemmes.vercel.app` or `https://lesfemmes.io`.
